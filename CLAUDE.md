# Lead Discovery Agent

## Om projektet
Det här projektet tillhör Tobias, konsult inom organisationsutveckling, ledarskap och handledning med utgångspunkt i gestaltteorin. Han hjälper individer och grupper att utveckla soft skills för att navigera AI-transformationen.

## Teknik
- Byggs med HTML, CSS och JavaScript (ingen React, inget byggsteg)
- Koden sparas på GitHub
- Webbsidan publiceras med Vercel
- Data sparas i Supabase

## Idealklient
- Organisationer/företag som ser utmaningar med AI-transformationen
- Vill ha samsyn om hur man ska jobba med AI
- Nyfikna på att utforska det nya, värdesätter det relationella
- Inser att man behöver bli bättre på att jobba ihop för att styra AI-transformationen (inte låta tekniken styra)
- Alla storlekar, alla branscher (främst de mest påverkade av AI just nu)
- Främst Stockholmsområdet, men hela Sverige om relevant

## Sökstrategier
- Företag som annonserar om AI-transformation, change management eller organisationsutveckling
- LinkedIn-profiler: HR-chefer, L&D-ansvariga, CTO:er
- Nyhetsartiklar om svenska företag mitt i AI-omställning
- Konferenser och events: TechSverige, AI Sweden, HR-summit
- Företag som söker roller som förändringsledare eller AI-ansvarig
- Företag som uttalar sig om AI-strategi eller digital transformation
- Nyligen omorganiserade företag

## Design
Ren och modern — mycket luft, minimalt, ljusa färger. Enkel typografi, tydliga kontraster.
